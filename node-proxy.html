Setup a NodeJS Proxy Service

Execution Steps

Run the below command:

kubectl run nodepod --image=node:alpine -it -n <namespace>

This will create a new POD nodepod in the namespace specified by <namespace>

Next in the shell of nodepod, launch the NodeJS REPL

$> node

In the REPL, type .editor and press Enter to get into the Editor Mode.
Paste the code snipped and then press Ctrl+D to execute the code.

This will start the NodeJS Proxy Service at the specified port (default 8080 in below code snippet)

Once the NodeJS Proxy Service is started, create a port-forward and then you can connect to the service from your local machine.

NodeJS Proxy Service (Code Snippet)

Update the targetHost & targetPort values to match your target service's host and port values.


const net = require('net');

const proxyPort = 8080;
const targetHost = '10.26.0.5';
const targetPort = 1521;

// Create the proxy server
const proxyServer = net.createServer((clientSocket) => {
  console.log(`Client connected from: ${clientSocket.remoteAddress}:${clientSocket.remotePort}`);

  // Connect to the target server
  const targetSocket = net.connect(targetPort, targetHost, () => {
    console.log(`Connected to target: ${targetHost}:${targetPort}`);
  });

  // Pipe data from the client to the target
  clientSocket.pipe(targetSocket);

  // Pipe data from the target to the client
  targetSocket.pipe(clientSocket);

  // Handle errors and closing connections
  clientSocket.on('error', (err) => {
    console.error('Client socket error:', err);
    targetSocket.end();
  });

  targetSocket.on('error', (err) => {
    console.error('Target socket error:', err);
    clientSocket.end();
  });
  
  clientSocket.on('end', () => {
    console.log('Client disconnected.');
    targetSocket.end();
  });
});

proxyServer.listen(proxyPort, () => {
  console.log(`TCP proxy listening on port ${proxyPort}`);
});
